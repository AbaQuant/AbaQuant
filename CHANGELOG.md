# Changelog

All notable changes to AbaQuant are documented here. The project follows
[Semantic Versioning](https://semver.org/).

## [Unreleased]

### Added

- Release validation for Python 3.10 through 3.13.
- Minimum-dependency, wheel, source-distribution, and PyPI-description checks.

## [1.0.0rc1] - Unreleased

### Added

- Stable domain facades for financial mathematics, derivatives, credit,
  portfolios, rates, market data, visualization, reports, risk, and provenance.
- Deterministic tests, runnable examples, and a complete Sphinx API reference.
- Optional Yahoo Finance, SEC/XBRL, FRED, Matplotlib, and Plotly workflows.

### Changed

- Reorganized pre-v1 modules into the public `abaquant.derivatives`,
  `abaquant.credit`, and `abaquant.portfolio` namespaces.
