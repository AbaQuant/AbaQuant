abaquant.portfolio
==================

**Import path:** ``abaquant.portfolio``

**Domain:** Portfolio construction, optimization, backtesting, risk metrics, and stress testing.

Package purpose
---------------

Portfolio estimation contexts, constraints, and specialized allocation families.

How to use this package
-----------------------

Defines the package facade and supported import surface. Use this package to transform return histories and covariance estimates into weights, then evaluate those weights out of sample and under explicit scenarios.
