abaquant.marketdata.ticker
==========================

**Import path:** ``abaquant.marketdata.ticker``

**Domain:** Provider-neutral market-data facades, normalized records, caching, and analytics.

Purpose
-------

Lazy single-ticker applied market-data interface.

When to use it
--------------

Use this package to retrieve or inject quotes, price history, option chains, and financial statements while preserving a stable analytical interface.

Public objects
--------------

* **function:** ``get_ticker`` ‚Äî Create a lazy applied interface for one normalized ticker symbol.
* **class:** ``MarketTicker`` ‚Äî Lazy market-data facade with immutable identity and mutable session state.
  * ``MarketTicker.spot`` ‚Äî Return the latest available spot-like quote supplied by the configured provider.
  * ``MarketTicker.dividend_yield`` ‚Äî Return the provider dividend yield or the documented fallback.
  * ``MarketTicker.visualize`` ‚Äî Return a market-price history figure for this ticker.
* **class:** ``TickerHistory`` ‚Äî Historical-price and realized-volatility namespace for a market ticker.
  * ``TickerHistory.prices`` ‚Äî Return the normalized price data required by this interface.
  * ``TickerHistory.realized_volatility`` ‚Äî Estimate trailing historical realized volatility from ticker prices.
* **class:** ``TickerOptionAnalytics`` ‚Äî Listed-option retrieval and model-analytics namespace for one ticker.
  * ``TickerOptionAnalytics.expirations`` ‚Äî Return the available listed option expiration dates.
  * ``TickerOptionAnalytics.chain`` ‚Äî Return a normalized listed option chain for one expiration.
  * ``TickerOptionAnalytics.analytics`` ‚Äî Return listed-option-chain analytics for one expiration.
  * ``TickerOptionAnalytics.bsm`` ‚Äî Price a European option under Black--Scholes--Merton using applied ticker inputs.
  * ``TickerOptionAnalytics.greeks`` ‚Äî Return the model sensitivities implemented by this model.
  * ``TickerOptionAnalytics.listed_implied_volatility`` ‚Äî Retrieve the listed implied volatility of the nearest available contract.
  * ``TickerOptionAnalytics.solve_implied_volatility`` ‚Äî Solve the inverse Black--Scholes--Merton problem for implied volatility.
  * ``TickerOptionAnalytics.compare_models`` ‚Äî Compare the option prices generated by the available pricing models.
* **class:** ``TickerFundamentalData`` ‚Äî Lazy fundamental-statement retrieval namespace for one ticker.
  * ``TickerFundamentalData.info`` ‚Äî Return provider metadata normalized to a plain Python dictionary.
* **class:** ``TickerCreditMetrics`` ‚Äî Manual fundamental credit-proxy namespace for one ticker.
  * ``TickerCreditMetrics.assess`` ‚Äî Evaluate manually supplied fundamentals for the ticker.
  * ``TickerCreditMetrics.assess_from_financials`` ‚Äî Build provider-fed credit inputs from cached statements and assess them.
  * ``TickerCreditMetrics.proxy_metrics`` ‚Äî Return flat manual credit-proxy metrics for convenient inspection.
  * ``TickerCreditMetrics.synthetic_score`` ‚Äî Return the 0--100 heuristic synthetic credit-proxy score.
  * ``TickerCreditMetrics.altman_z_score`` ‚Äî Return the traditional public-company Altman Z-score when available.
  * ``TickerCreditMetrics.piotroski_f_score`` ‚Äî Return the complete nine-signal Piotroski F-score when available.
