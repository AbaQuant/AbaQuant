abaquant.marketdata
===================

**Import path:** ``abaquant.marketdata``

**Domain:** Provider-neutral market-data facades, normalized records, caching, and analytics.

Package purpose
---------------

Lazy applied market-data interfaces and provider-independent analytics.

How to use this package
-----------------------

Defines the package facade and supported import surface. Use this package to retrieve or inject quotes, price history, option chains, and financial statements while preserving a stable analytical interface.
