abaquant.reports
================

**Import path:** ``abaquant.reports``

**Domain:** Structured analytical reports and Markdown, HTML, or lightweight PDF export.

Package purpose
---------------

Exportable report primitives and report builders.

How to use this package
-----------------------

Defines the package facade and supported import surface. Use these objects after calculations are complete and results must be packaged for review, storage, or distribution.
